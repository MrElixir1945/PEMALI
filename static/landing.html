The Reasoning Loop

PEMALI tidak sekadar menjalankan perintah statis. Menggunakan arsitektur ReAct (Reasoning and Acting), agen ini berpikir layaknya auditor profesional: menarik data satelit, menganalisis vegetasi, dan mencocokkan zonasi hukum sebelum menarik kesimpulan akhir.

INPUT → THOUGHT → ACTION
agent_thought "Lokasi yang diminta adalah Gianyar. Saya belum tahu kondisi riil di sana. Saya harus memanggil satellite_intelligence untuk mendapatkan data tutupan awan dan citra lahan terbaru sebelum menganalisis kerusakan."
tool_execution { "tool_name": "satellite_intelligence", "parameters": { "location": "Gianyar" } }

Pillars of Tri Hita Karana

🌱

Palemahan

Evaluasi ekologi murni. Mendeteksi penyusutan ruang terbuka hijau, deforestasi, dan menjaga keseimbangan lanskap alam Bali.

👥

Pawongan

Analisis sosial kemasyarakatan. Memverifikasi regulasi peruntukan lahan (OSINT) untuk meminimalisasi konflik tata ruang antara pemodal dan warga lokal.

🙏

Parahyangan

Harmoni spiritual. Memastikan area suci dan zona pelestarian pura tidak terlanggar oleh laju pembangunan infrastruktur berlebihan.
